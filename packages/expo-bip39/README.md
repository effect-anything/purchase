# @effect-x/expo-bip39

Expo module bindings for BIP39 mnemonic helpers.

## Install

```bash
pnpm add @effect-x/expo-bip39 effect expo-modules-core
```

## Entrypoints

- `@effect-x/expo-bip39`

## Notes

- This package is intended for Expo and React Native environments.
- Consumers should follow the normal Expo modules setup for native projects.
